Mon Thème Classique — WordPress
Installation:
1) Apparence → Thèmes → Ajouter → Téléverser → importer le ZIP.
2) Activer le thème.
3) Aller dans Réglages → Lecture pour choisir la page d’accueil (si nécessaire).

Menus:
- Apparence → Menus → créer 'Menu principal' et l’assigner à l’emplacement 'Menu principal'.
- Idem pour 'Menu pied de page'.

Personnalisation:
- Ajoutez vos styles dans style.css.
- Modifiez header.php et footer.php pour adapter le layout.
