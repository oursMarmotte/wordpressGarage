=== Screenshot Theme ===
Conçu pour WordPress 6.5+ (thème bloc / FSE).

Installation
1. Téléchargez le fichier ZIP du thème et installez-le via Apparence → Thèmes → Ajouter → Téléverser.
2. Activez le thème.
3. Allez dans Éditeur de site pour personnaliser les templates, couleurs et typographies.

Personnalisation à partir de votre capture d'écran
- Remplacez le pattern 'Hero' et/ou créez de nouveaux patterns pour imiter l'agencement (sections, colonnes, cartes, etc.).
- Adaptez palette/corner radius/espacements dans theme.json.
- Besoin d'un pixel-perfect ? Envoyez la capture et j'alignerai header, footer, grilles, etc.
